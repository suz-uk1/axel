Recommended columns

  1. Vendor name
  2. Service / purpose
  3. Data processed (PII? customer data? secrets?)
  4. Access level (prod? admin?)
  5. Criticality / tier (High/Med/Low)
  6. Inherent risk (simple 1–5 or tier)
  7. Assurance evidence (SOC 2, ISO cert, pen test, questionnaire)
  8. Contract clauses (security, breach notification, sub-processors)
  9. Owner (business + security)
  10. Last review date + next review date
  11. Evidence link (folder / ticket / doc)
