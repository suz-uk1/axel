The evidence list (12 artifacts)

  1. Information Security Policy (versioned, last updated date)
  2. Statement of Applicability (SoA) and control ownership
  3. Risk assessment: risk register + treatment plan
  4. Access management: onboarding/offboarding + access review record
  5. Logging & monitoring: what’s logged + retention + review cadence
  6. Change management: PR approvals + release log
  7. Incident management: process + one drill/tabletop record
  8. Vulnerability management: scanning + remediation SLAs + sample tickets
  9. Supplier security: vendor list + due diligence evidence
  10. Backup/BCP: backup policy + restore test evidence
  11. Internal audit: checklist + latest findings/actions
  12. Management review: minutes + actions

How to answer fast

For each artifact, provide: owner role, evidence link, last updated date. If you do this in a spreadsheet, procurement can verify quickly.
