# NIS2 Micro Offer — Incident Reporting Timeline Kit (v0)

## What this is
A tiny, pragmatic kit to help teams set up NIS2-style incident reporting timestamps and workflow evidence.

## Files
- `incident-reporting-timeline.csv` — timeline + required fields
- `incident-notification-draft.md` — email draft to authorities/customers (template)
- `how-to-use.md` — 20-minute setup guide
