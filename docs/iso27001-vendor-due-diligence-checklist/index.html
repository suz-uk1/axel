ISO 27001 Vendor Due Diligence Checklist

Use this checklist when enterprise customers ask for vendor security due diligence evidence.

Start with free sheet: Evidence Map (CSV)

Request these 10 items

  1. Information Security Policy (latest version)
  2. Statement of Applicability (SoA)
  3. Access review record
  4. Incident response process + latest drill
  5. Vulnerability management cadence + sample remediation proof
  6. Supplier risk process
  7. Backup and restore test record
  8. Internal audit summary
  9. Management review actions
  10. Roles/owners for each control area

Fast response format

For each request: owner role, evidence link, last updated date, review cadence.
