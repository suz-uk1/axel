Quality bar (the real test)

  • Every evidence item has a named owner
  • Every link opens quickly (no permission surprises)
  • Every screenshot/log has a timestamp
  • You can answer follow-ups without scavenger hunts

Fast setup (10 minutes)

  • Create one folder: ISMS/Evidence/Stage2/
  • Create one sheet: Evidence Map (controls → evidence → owner → link)
  • Pick top 20–30 controls in scope and assign owners

Checklist

Access control

  • Joiner/mover/leaver evidence (ticket/HR workflow)
  • MFA enforcement evidence
  • Privileged access evidence (admin groups, reviews)
  • Periodic access review evidence

SDLC / change management

  • Code review rules + approvals enforced
  • CI/CD pipeline logs retained
  • Vulnerability workflow + fix SLAs (even basic)

Logging / monitoring / incidents

  • Central logging configuration + retention
  • Alert rules + on-call process
  • Incident tickets + postmortem template evidence

Backup / recovery

  • Backup configuration evidence
  • Restore test evidence (date + result)
  • RTO/RPO documented and tied to risk

Assets & suppliers

  • Asset inventory (systems + data stores)
  • Vendor list + security review evidence
  • Key supplier contract pointers / DPA links

Governance glue

  • SoA is consistent with scope + risk
  • Internal audit plan + at least one internal audit record
  • Management review minutes with decisions + actions
