Evidence Index Template (CSV)

This is the fastest way to kill the “SoA says yes, but where’s the proof?” loop. Keep one row per control with: owner, cadence, evidence link, and last run date.

Download CSV TRACE checklist Toolkit checkout ($199)

Tip: Put links to tickets/logs/reports/runbooks—not screenshots.

How to use (5 minutes)

  1. Pick your top 10 controls (from your SoA) that map to your top risks.
  2. For each, set owner + cadence and link one artifact that proves it runs.
  3. In internal audit, sample rows and verify evidence matches reality.

Columns

control_id, control_name, soa_applicable, soa_justification,
owner, team, cadence, last_run_date, evidence_link, evidence_type,
system_scope, notes
