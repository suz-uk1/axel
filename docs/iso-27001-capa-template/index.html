Columns that actually work

  1. Finding / nonconformity
  2. Root cause (not symptom)
  3. Corrective action (what changes)
  4. Owner
  5. Due date
  6. Status
  7. Verification method
  8. Verification evidence link
  9. Closed date

Common CAPA failure mode

Without a verification evidence column, CAPA becomes “we’ll do it later”. Auditors want proof the corrective action was implemented and is effective.
