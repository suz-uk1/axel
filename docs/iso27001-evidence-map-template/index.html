Template columns

  • Control (e.g., A.5.15)
  • Evidence item (what you’ll show)
  • Owner (a named person)
  • Link (opens instantly)
  • Last updated (timestamp)

Quality bar

  • If the auditor can’t open it, it doesn’t exist.
  • If it has no timestamp, it’s questionable.
  • If it has no owner, it won’t stay current.

How to use (10 minutes)

  1. Create a folder: ISMS/Evidence/Stage2/
  2. Create a Google Sheet called “Evidence Map”.
  3. Pick the top 20–30 controls in your scope.
  4. Assign owners and paste links for what you already have.
  5. Run a 7‑day sprint to fill the gaps.
