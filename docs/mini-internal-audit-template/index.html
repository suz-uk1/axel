Mini Internal Audit Template (CSV)

Stage 2 auditors sample controls. You should do the same—early. This template lets you run a 30–60 minute mini internal audit across 5–10 controls and log findings + CAPA owners/dates.

How to run (simple)

  1. Select 5–10 controls from your SoA that map to your top risks.
  2. For each control, define what “expected evidence” looks like.
  3. Follow the evidence link and verify it’s timestamped + reproducible.
  4. Log pass/fail and open 1–3 CAPAs with owners + due dates.

Columns

audit_id, audit_date, scope,
control_id, control_description, expected_evidence,
evidence_link, pass_fail, findings,
capa_owner, capa_due_date, notes
