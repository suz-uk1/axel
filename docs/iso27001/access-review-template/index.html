Recommended cadence (startup)

  • Quarterly: production/admin access + cloud console access
  • Monthly: employee offboarding checks (quick)
  • Event-driven: role changes, incidents, major org changes

Review checklist (copy/paste)

  • Any ex-employees/contractors still have access?
  • Any privileged accounts without MFA?
  • Any shared accounts still in use?
  • Any users with access not needed for their role?
  • Any stale service accounts / tokens?
  • Evidence captured (export/screenshot) + stored location noted

Auditor tips
